# Rajvi Zala - Portfolio

A responsive portfolio website built with Next.js 15, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy on Vercel

1. Push to GitHub
2. Import repo at [vercel.com](https://vercel.com)
3. Deploy (Next.js auto-detected)

Optional: Add `GITHUB_TOKEN` environment variable for Proof of Work (GitHub PRs) section.

## Project Structure

- `app/` - Pages and layout
- `components/` - UI components and sections
- `lib/` - Data and utilities
